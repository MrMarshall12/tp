### Project: ServeMate

ServeMate is a desktop address book created for **administrative staff** for tingkat food delivery to manage their customer contacts for delivery planning. The user interacts with it using a CLI, and it has a GUI created with JavaFX.

Given below are my contributions to the project.

* **New Feature**: `unschedule` command ([\#80](https://github.com/AY2526S2-CS2103T-W14-2/tp/pull/80)).
    * What it does: Allow the user to unschedule deliveries tied to a particular customer.
    * Justification: This feature is important as it allows tingkat caterers to help customers cancel their tingkat subscription plans, while keeping their records intact.
    * Highlights: This was the first command implemented that explicitly interacts with a customer's delivery. As such, I had to ensure that future commands interact correctly with the `Delivery` and `Person` classes.

* **Enhancements to Existing Features**:
